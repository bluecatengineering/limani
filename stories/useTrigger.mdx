# useTrigger

Returns a stateful boolean and a function to toggle it.

## Returns

`useTrigger~Return` - Tuple which can be used to control a trigger or any other useful purpose.

## Example

```js
import {useTrigger} from '@bluecateng/limani';

const Example = () => (
	const [trigger, setTrigger] = useTrigger()

    return (
        <button onClick={setTrigger}>{trigger ? 'Toggled' : 'Click to trigger Toggle'}</button>
    );
);
```

## Other types

### useTrigger~Return

`Array` - return type for `useTrigger`.
