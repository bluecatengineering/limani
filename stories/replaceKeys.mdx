# replaceKeys

Replaces the key of inputData using the provided keyMapping and returns it.

## Parameters

<table>
    <thead>
        <tr>
            <th>Param</th>
            <th>Type</th>
            <th>Description</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>inputData</td>
            <td>`object`</td>
            <td>Object containing key to be looked up and its value</td>
        </tr>
        <tr>
            <td>keyMapping</td>
            <td>`object`</td>
            <td>
                Object containing keys which has to be looked up and their
                corresponding replacements
            </td>
        </tr>
    </tbody>
</table>

## Returns

`replaceKeys~Return` - Object containing replaced key and same value as inputData

## Example

```js
import { replaceKeys } from '@bluecateng/limani';

const groupFieldsMapping = {
    'group': 'groupName',
    'landing_page': 'landingPage',
    'show_bam': 'showBAM',
    'show_help': 'showHelp',
};

const Example = () => (
    const inputData = {
        'show_help': true
    }

	const output = replaceKeys(inputData, groupFieldsMapping);
    ...
);
```
