# usePageModalSpinner

Displays a spinner while waiting for a process to complete.

## Returns

`usePageModalSpinner~Return` - Returns a spinner while waiting for data or process to load.

## Example

Use the `SimplePage` component to wrap it inside. The `SimplePage` component will handle rendering of the `PageModalSpinner` component
when the child component needs to wait for a process to complete.
The `setBusy` attribute can be used to show the spinner on the page.
When the value is set to TRUE the spinner is displayed, set it FALSE to not display the spinner.

```js
import { Button } from '@bluecateng/pelagos';
import { SimplePage, usePageModalSpinner } from '@bluecateng/limani';

function Example() {
    const { setBusy } = usePageModalSpinner();
    const handleButtonClick = () => {
        setBusy(true);

        // Simulating an asynchronous task
        setTimeout(() => {
            setBusy(false);
        }, 2000);
    };
    return <Button text='Trigger Spinner' onClick={handleButtonClick} />;
}

export default function App() {
    return (
        <SimplePage>
            <Example />
        </SimplePage>
    );
}
```
