# useLanguage

Returns current language returned from Platform data API and
a boolean denoting if the data is being fetched currently.
The platform data should be fetched by Limani before this hook can be used.

## Returns

`Object` - Contains language fetched from the Plaform data API.

## Example

```js
import { useLanguage } from '@bluecateng/limani';

const Example = () => (
    const { language } = useLanguage()
    ...
);
```

## Other types

### Return

`Object` - Object containing language string.
