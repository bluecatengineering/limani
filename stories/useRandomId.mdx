# useRandomId

Returns either the specified string or a stable random string.
Random string is returned when no argument is passed.
Specified string is returned when it's passed as an argument.

## Parameters

<table>
    <thead>
        <tr>
            <th>Param</th>
            <th>Type</th>
            <th>Description</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>id</td>
            <td>`string`</td>
            <td>Input Id to use.</td>
        </tr>
    </tbody>
</table>

## Returns

`useRandomId~Return` - Returns either the specified string or a stable random string.

## Example

```js
import {useRandmonId} from '@bluecateng/limani';

/** Generates a stable random string */
const RandomIdExample = () => (
	const randomId = useRandmonId()
    ...
);

/** Returns the specified string */
const ConstantIdExample = (id) => (
	const specifiedId = useRandmonId(id)
    ...
);
```
