# resetForm

triggers the reset function from @bluecateng/auto-forms on change to initialFormData.
Ensures that the formData is up to date whenever element is rendered.

## Parameters

<table>
    <thead>
        <tr>
            <th>Param</th>
            <th>Type</th>
            <th>Description</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>initialFormData</td>
            <td>`object`</td>
            <td>Form Data object to be monitored for changes</td>
        </tr>
    </tbody>
</table>

## Example

```js
import { resetForm } from '@bluecat/limani';

/** ensures the form is reset in case of change in initialFormData */
const ExamplePageObject = ({initialFormData }) => (
	resetForm(initialFormData);
    ...
);
```
